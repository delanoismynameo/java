# Project Title (Piano Improvisation App)

This app gives the user a random key and records the time they started improvising.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Make sure you have Java JDK on the computer
2. Run on VS Code by using the play button

Instructions for using the software:

1. Type 1 in the terminal
2. Practice in the provided key. Once you are done press enter.
3. Press 2 to view the practice log, or press 3 to exit the application

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Install Java JDK 8 or higher
